# 🛒 Online Auction System

A Full Stack Online Auction System developed using **React, Spring Boot, and MySQL**. Users can register, log in, add auction products, browse available products, and place bids.

---

## 🚀 Features

- User Registration
- User Login
- Product Listing
- Product Details
- Add Products
- Place Bids
- Dashboard
- Responsive UI

---

## 🛠 Tech Stack

### Frontend
- React
- Vite
- Bootstrap
- Axios

### Backend
- Java
- Spring Boot
- Spring Data JPA

### Database
- MySQL

## ▶️ How to Run

### Backend

```bash
cd online-auction-system\ backend
./mvnw spring-boot:run
```

### Frontend

```bash
cd "online auction system frontend"
npm install
npm run dev
```
