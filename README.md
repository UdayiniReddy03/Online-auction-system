# 🛒 Online Auction System

A Full Stack Online Auction System developed using **React, Spring Boot, and MySQL**. Users can register, log in, add auction products, browse available products, and place bids.

---

## 🚀 Features

- User Registration
- User Login
- Product Listing
- Product Details
- Add Products
- Place Bids
- Dashboard
- Responsive UI

---

## 🛠 Tech Stack

### Frontend
- React
- Vite
- Bootstrap
- Axios

### Backend
- Java
- Spring Boot
- Spring Data JPA

### Database
- MySQL

---

## 📂 Project Structure

```
Online auction system
│
├── online auction system frontend
└── online-auction-system backend
```

---

## ▶️ How to Run

### Backend

```bash
cd online-auction-system\ backend
./mvnw spring-boot:run
```

### Frontend

```bash
cd "online auction system frontend"
npm install
npm run dev
```
## Database Setup

Create the database:

```sql
CREATE DATABASE auction_db;
```

Update `application.properties` with your own MySQL username and password before running the backend.
